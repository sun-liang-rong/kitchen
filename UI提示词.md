# 厨房许愿池 Stitch UI 生成提示词

## 使用说明

将下面的“总提示词”复制到 Stitch 中，用于生成完整 App UI。

如果 Stitch 对一次性生成多页面支持不好，可以使用后面的“分页面提示词”逐页生成。

## 总提示词

请为一款移动端 App 生成高保真 UI 设计，产品名为 **厨房许愿池**。

这是一款面向情侣、夫妻、双人小家庭的共享吃饭愿望 App。双方都可以把“想吃什么”投进共享许愿池，也都可以认领对方的愿望，通过实现、换菜、延期、一起做、先搁置等方式完成一次吃饭协商。

产品核心不是“点菜”，而是：

「我表达想吃什么，你可以选择怎么回应；你愿意为我实现，我也愿意搭把手。」

请设计一套完整的 iOS/Android 风格移动端 App UI，重点体现温柔、生活感、情侣协商、轻量不算账的产品气质。

## 视觉风格

- 设计风格：温柔、干净、生活化、轻量高级，不要幼稚卡通，不要像外卖 App，不要像任务管理工具。
- 目标用户：同居情侣、夫妻、双人小家庭。
- 情绪关键词：温暖、被看见、好商量、轻松、有烟火气。
- 主色建议：温暖珊瑚橘、奶白、浅米色、鼠尾草绿、柔和蓝灰。
- 背景：浅暖色背景，卡片使用白色或暖白色。
- 圆角：卡片和按钮圆角控制在 8px 左右，不要过度胶囊化。
- 字体：中文移动端无衬线字体，标题清晰，正文易读。
- 图标：使用简洁线性图标，例如碗、锅、日历、心愿、列表、时钟、对话、餐具。
- 组件要克制，避免大面积渐变、装饰球、复杂插画。
- 页面第一眼要像一个真实可用的 App，而不是营销落地页。

## 设计原则

- 用“许愿”替代“点菜”。
- 用“回应、换个版本、做轻松版、改天实现、一起做、先放池子里”替代“接单、拒绝、任务取消”。
- 用“搭把手”替代“家务交换”。
- 不出现积分、排行榜、谁欠谁、强制公平统计。
- 文案要像情侣之间商量，不要像系统派任务。
- 所有核心操作要轻，不要让用户感觉像填复杂表单。

- 状态流转新增：
  - 她确认了：同意今晚换成红烧鸡腿。
- 详情页底部出现主按钮：
  - 记录兑现
- 愿望卡上显示：
  - 已认领
  - 她刚刚确认
  - 同意换成红烧鸡腿

- 操作：继续回应 / 先放池子里

## 分页面提示词

### 1. 许愿池首页提示词

为移动端 App「厨房许愿池」设计首页。页面包含双人绑定状态、双方厨房状态、横向筛选 tab、愿望卡列表、底部导航和悬浮许愿按钮。风格温柔、生活化、轻量高级，不像外卖 App，不像任务管理工具。愿望卡需要展示许愿人、愿望强度、当前状态、标题、描述、希望时间、最近回应、搭把手标签和操作按钮。筛选 tab 包括全部、她许的、我许的、池中、待确认、已认领、改天、已兑现。

### 2. 发起许愿页提示词

为移动端 App「厨房许愿池」设计发起许愿页面。页面标题“许个愿”，副标题“说出想吃的，也带上一点搭把手”。包含输入框“想吃什么”、想要的感觉 chips、希望时间 chips、愿望强度 chips、可接受替代 chips、我愿意搭把手 chips、备注文本框和主按钮“丢进许愿池”。整体轻量，不像复杂问卷。

### 3. 回应愿望底部抽屉提示词

为「厨房许愿池」设计回应愿望的底部抽屉。标题“回应她的愿望”。展示对方愿望“可乐鸡翅”和她愿意洗碗、拖地。提供回应选项：今晚我来实现、做轻松版、换成红烧鸡腿、改天实现、一起做吧、先放池子里。提供改天时间、原因标签、备注输入框和主按钮“发给她”。强调温柔协商，不使用拒绝这个词。

### 4. 愿望详情页提示词

为「厨房许愿池」设计愿望详情页。页面展示愿望主卡、等待她确认卡、状态流转和回应历史。愿望为“可乐鸡翅”，状态“待她确认”。展示你提议“换成红烧鸡腿”，原因“家里没有鸡翅但有鸡腿”。提供两个按钮：“她同意了”“她想再改”。状态流转包括她许下愿望、你回应了愿望、等待她确认。

### 5. 兑现记录底部抽屉提示词

为「厨房许愿池」设计饭后兑现底部抽屉。标题“记录这次兑现”，说明“不用打分，只留下对下次有用的记忆”。包含实际吃了什么、谁实现了这个愿望、谁搭了手、饭后反馈 chips、加入我们家的常吃菜开关和主按钮“愿望已兑现”。不要使用星级评分。

### 6. 我们家的菜页提示词

为「厨房许愿池」设计“我们家的菜”页面。用卡片网格展示番茄鸡蛋面、红烧鸡腿、酸辣土豆丝、番茄牛腩。每张卡展示适合时间、难度、谁会做和最近反馈。增加“适合今天”的推荐卡：番茄鸡蛋面 + 凉拌黄瓜，少洗锅，20 分钟内吃上。

### 7. 兑现记录页提示词

为「厨房许愿池」设计兑现记录页。页面展示最近 7 天的饭桌记忆列表。记录包括“刚刚兑现：红烧鸡腿”“我实现 · 她洗碗和拖地 · 已加入常吃菜”等。风格像生活记录，不像评分或任务日志。
